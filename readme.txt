Скрипт task_7.1.py - создает файл стартера проекта "blank_project.json", далее по этому файлу создает стартер проекта.
Скрипт принимает один параметр - название стартера проекта (файл должен быть уже создан).

Скрипт task_7.2.py - создает файл стартера проекта "config.json", далее по этому файлу создает стартер проекта.

Скрипт task_7.3.py - собирает в папке проекта все шаблоны 'templates', далее создает отдельную папку 'templates'
в корне проекта со всеми шаблонами. Скрипт принимает один параметр - имя директории проекта.

Скрипт task_7.5.py - создает файл {name_folder}_summary.json, который хранит в себе словарь с ключами - верхняя
граница размера файла и значениями (<files_quantity>, [<files_extensions_list>])
